itao 爱淘
====
自动领取淘金币的chrome扩展

逻辑规则
====
* 如果用户没有输入用户信息保存，则不尝试进行登录。仅在用户手动登录了淘宝的情况下尝试获得淘金币
	+ 当用户退出当前登录的淘宝账户，badge清除显示淘金币信息。
	
* 如果用户输入了登录信息
	+ 当前没有登录淘宝。则当用户打开一个新的tab的时候程序尝试自动登录淘宝。
	+ 当前登录了淘宝，则badge显示当前登录用户的淘金币数目。
	+ 当用户退出当前登录的淘宝账户，badge显示保存用户的淘金币信息。
	
* 仅在上次登录的用户是保存用户的时候才尝试自动登录情况（退出操作会导致用户下次登录的时候需要验证码？）
	+ 如果登录失败则一天仅提示用户一次（需要用户点击页面顶部的不再提示按钮）
